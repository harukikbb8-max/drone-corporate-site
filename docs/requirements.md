# 要件定義書：ドローン測量会社 コーポレートサイト

---

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | ドローン測量会社 コーポレートサイト新規構築 |
| クライアント | ドローン測量会社（新規設立） |
| 目的 | 企業の信頼性向上、サービス認知拡大、問い合わせ獲得 |
| 公開予定日 | 2026年3月末 |
| 構築方式 | HTML/CSS/JavaScript（静的サイト） |

---

## 2. ターゲットユーザー

| 項目 | 内容 |
|------|------|
| 年齢層 | 20〜40代 |
| 属性 | ビジネスマン（建設・土木・農業関連の意思決定者） |
| 利用デバイス | PC 60% / スマートフォン 35% / タブレット 5%（想定） |
| 利用シーン | ドローン測量サービスの比較検討・問い合わせ |

---

## 3. デザイン要件

### 3.1 カラースキーム

| 用途 | カラーコード | 色名 |
|------|-------------|------|
| プライマリ | #FF6B00 | オレンジ |
| セカンダリ | #1B2A4A | 紺色（ダークネイビー） |
| アクセント | #FF8C33 | ライトオレンジ |
| 背景（メイン） | #FFFFFF | ホワイト |
| 背景（サブ） | #F5F7FA | ライトグレー |
| テキスト（メイン） | #2D2D2D | ダークグレー |
| テキスト（サブ） | #6B7280 | ミディアムグレー |

### 3.2 デザインコンセプト

- **シンプル＆プロフェッショナル**: 大林組のような洗練された企業サイト
- **先進性**: ドローンテクノロジーの先端イメージを演出
- **信頼感**: 紺色ベースで信頼性と安定感を表現
- **アクション誘導**: オレンジのCTAで問い合わせへ誘導

### 3.3 フォント

| 用途 | フォント |
|------|---------|
| 見出し（日本語） | Noto Sans JP (700) |
| 本文（日本語） | Noto Sans JP (400) |
| 英字・数字 | Inter |

### 3.4 アニメーション要件

- スクロール連動のフェードイン・スライドアップ（Intersection Observer API使用）
- ヒーローセクション：テキストのタイプライター風出現
- ホバーエフェクト：カード要素の浮き上がり・シャドウ変化
- ページ遷移：フェードトランジション
- カウントアップアニメーション（数値実績セクション）
- パーティクル背景（ヒーローセクション）

### 3.5 レスポンシブ対応

| ブレイクポイント | デバイス |
|----------------|---------|
| 〜767px | スマートフォン |
| 768px〜1023px | タブレット |
| 1024px〜 | デスクトップ |

## 7. 技術要件

| 項目 | 仕様 |
|------|------|
| HTML | HTML5 セマンティックマークアップ |
| CSS | CSS3（カスタムプロパティ使用）|
| JavaScript | Vanilla JS（外部ライブラリ不使用） |
| フォント | Google Fonts（Noto Sans JP, Inter） |
| 画像 | SVGイラスト（自作）+ フリー素材（Unsplash等） |
| SEO | メタタグ、OGP、構造化データ |
| パフォーマンス | 画像遅延読込、CSS/JSの最適化 |

---

## 8. 画像素材方針

| 方針 | 詳細 |
|------|------|
| フリー素材 | Unsplash（商用利用無料・クレジット不要）を使用 |
| SVGイラスト | アイコン・装飾はSVGで自作 |
| プレースホルダー | 実写真未提供箇所はグラデーション背景+テキストで代替 |
| 著作権 | 全素材が商用利用可・著作権フリーであることを確認 |

---

## 9. 非機能要件

| 項目 | 要件 |
|------|------|
| ブラウザ対応 | Chrome, Safari, Firefox, Edge（最新版） |
| 表示速度 | LCP 2.5秒以内目標 |

| セキュリティ | 静的サイトのため最小限（HTTPS推奨） |
